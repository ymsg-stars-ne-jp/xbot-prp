# X BOT PRO Application

This repository contains the X BOT PRO application, which includes various functionalities to manage bots efficiently and securely.

## Setup

1. Clone the repository
2. Rename `.env.example` to `.env` and fill in the necessary environment variables.
3. Install dependencies.

## Directory Structure
- `config/` - Configuration files
- `includes/` - Including necessary functionalities
- `api/` - API endpoints
- `public/` - Public assets and entry points
- `src/` - Source code of the application

## Security Improvements
- All sensitive data is stored in environment variables.
- Proper input validation and error handling are implemented throughout the application.
